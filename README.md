# PythonPortfolio
|This is a collection of my favorite projects created in my Jones College Prep AP CSP class.

Project 1: create.py and presidents.csv
|Summary: Used for my AP Create task, this project uses a data set of all US presidents to provide a vast number of functions, including a menu interface as well. 
|Key features: Pairs data of presidents and many aspects of their presidency. Functions include portraits, term years, designation, and party platform. The menu incorporates these functions into a user-friendly interface to use the program as an application.

Project 2: calculator.py
|Summary: A limited calculator program where inputs of numbers and an operation are computed to provide math accurate to a calculator.
|Key features: Includes the basic operations (+, -, *, /) and limited to 2 number inputs. This program effectively performs basic operations in the correct order of operations.

Project 3: congestion.py
|Summary: Uses the data from Chicago streets and corresponding traffic volume to evaluate trafic volume.
|Key features: Can scan for the busiest street in the city. It can also filter for streets that meet a given traffic volume and print them.

Project 4: grocerystore.py
|Summary: Simulates the inventory of prices at a grocery store.
|Key features: Can create a budget by looking for item which could be purchased under a given limit. It can also apply a storewide discount and calculate a total.

Project 5: search.py
|Summary: Searches for a generated number using binary and linear searches.
|Key features: Includes both search types and can run both tests on one terminal execution. It not only searches but also compares the number of attempts based on a generated number.
